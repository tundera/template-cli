# Node.js CLI Template

## Usage

```sh
# env
TEMPLATE_LOGIN=<login>
TEMPLATE_TOKEN=<token>

# sh
npx template-cli
```

## Config

Support config file `template.config.js`, for more details read [types.ts](./types/index.ts).

```js
module.exports = {
  /* options */
}
```
