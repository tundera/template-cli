# Node.js CLI Template

## Usage

```sh
# env
PROGRAM_LOGIN=<login>
PROGRAM_TOKEN=<token>

# sh
npx template-cli
```

## Config

Support config file `cli.config.js`, for more details read [types.ts](./src/types.ts).

```js
module.exports = {
  /* options */
}
```
